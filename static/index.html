API Reference

Endpoints

GET /

Serves the static index page from the ./static directory using Go's built-in file server.
GET /form.html

Renders the HTML contact form with name, email, and address fields.
GET /hello

A simple text endpoint demonstrating handler functions and method validation.
POST /form

Accepts POST form submissions, validates fields, and returns a structured JSON response.

GET /api/health

Returns server status, uptime, and timestamp as JSON. Standard for load balancers.
Architecture

What's Inside

⚡
Zero Dependencies

Built entirely on Go's standard library. No third-party frameworks, no bloat.

🪵
Logging Middleware

Every request is logged with method, path, status code, and response duration.

🔒
Secure Timeouts

Configured read, write, and idle timeouts to protect against slow-client attacks.

🌐
JSON API

The /form and /api/health endpoints respond with structured JSON payloads.

📦
Go Modules

Uses go mod — the modern standard for dependency management in Go.

🚀
Deploy Ready

PORT env var support. Compile with go build into a single binary for any platform.
